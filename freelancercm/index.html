Apache APISIX

Cloud-native microservices API gateway, delivering the ultimate performance, security, open source and scalable platform for all your APIs and microservices.

APISIX is based on Nginx and etcd. Compared with traditional API gateways, APISIX has dynamic routing and plug-in hot loading, which is especially suitable for API management under micro-service system.

View on GitHub

Why Apache APISIX？

High availability, safety first

Built-in high-availability solution and multiple security plug-ins for identity authentication and interface verification, putting stability and security first.

Built-in Dashboard, powerful development interface

It comes with a simple, easy-to-use built-in Dashboard and a powerful and flexible interface for developers to use.

Ultimate performance

From routing, json schema, to built-in plug-ins, which are designed and implemented to ultimate performance requirements.

Multi-protocol and multi-platform support

Supports http to gRPC transcoding, supports websocket, gRPC, Dubbo, MQTT proxy, and supports multiple platforms including ARM64.
