Apache APISIX

云原生、高性能、可扩展的微服务 API 网关。

基于 Nginx 和 etcd 来实现，具备动态路由、插件热加载和 gRPC 代理和协议转换功能，更适合微服务体系下的 API 管理。

访问 GitHub

为什么选择 Apache APISIX？

高可用，安全第一

内置高可用方案，并提供多种身份认证和接口校验的安全插件，把稳定和安全放在第一位。

易用的管理界面，强大的开发接口

自带简洁、易用的管理界面，并提供强大、灵活的接口以便开发者使用。

极致性能

从路由、json schema 检验，到内置的丰富插件，都是按照最高的性能要求来设计和实现。

多协议和多平台支持

支持 http 到 gRPC 的协议转换，支持websocket、gRPC、Dubbo、MQTT 的代理，支持包括 ARM64 在内的多种平台。
